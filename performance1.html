การจำแนกประเภทภาพถ่ายด้วย Machine Learning

พัฒนาระบบที่สามารถจำแนกประเภทของภาพถ่ายได้โดยอัตโนมัติ เช่น การจำแนกภาพสัตว์, ภาพผลไม้ หรือภาพใบหน้า โดยใช้เทคนิค Machine Learning และ Deep Learning รวบรวมชุดข้อมูลภาพถ่าย (Dataset) เลือกและฝึกฝนโมเดล Machine Learning ที่เหมาะสม (เช่น Convolutional Neural Network - CNN) ประเมินผลประสิทธิภาพของโมเดล สร้างอินเทอร์เฟซ (Interface) ให้สามารถอัปโหลดภาพและรับผลการจำแนกได้

วันที่ : 15 มิถุนายน 2567

สถานที่ : ประเทศญี่ปุ่น

รางวัลระดับ : ชนะเลิศระดับนานาชาติ
